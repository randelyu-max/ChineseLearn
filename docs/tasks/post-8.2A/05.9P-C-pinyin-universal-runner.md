# Task 5.9P-C — 拼音接入通用 Session Runner

**优先级：** P0
**依赖：** 5.9P-B。
**目标：** 六种拼音题与汉字题共用正式 Session、离线、Outbox、反馈和完成流程。
**非目标：** 不录音、不做发音评分。

## 拼音 Tab

替换“一页六个 Demo”的产品结构：

- 显示当前拼音进度；
- 推荐“继续拼音学习”或“复习声调”；
- 启动正式 Session；
- Showcase 仅开发可达。

## Runner 接入

六种组件适配统一 Activity Renderer。不得在页面中复制评分、调号或 Evidence 逻辑。

## 音频

- 预取 Session 所需音频；
- 缓存按 content hash；
- 离线可播放；
- 播放失败可重试；
- 不启用麦克风权限；
- 不上传声音。

## 测试

- 每题型正式 Session；
- 混合汉字/拼音 Session；
- 声调/轻声；
- 离线音频；
- 重播次数；
- Hint 与 Evidence 一致；
- App 重启；
- Server mismatch；
- Accessibility；
- E2E 拼音 → skill → review。

## 验收

拼音 Tab 和正式学习 Session 都能真实更新拼音/声调掌握度与复习计划。
