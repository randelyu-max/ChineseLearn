## V1 locale

必须完整支持：

- `en-US`
- `zh-CN`

`zh-TW` 只有达到完整覆盖才显示；否则从 Onboarding 选择器和 Store 声明移除，但保留 Traditional Script 课程偏好。

## 架构

建立共享 message catalog、typed keys、插值、复数/时间接口、Locale Provider 和 fallback。选择现有栈兼容的实现并写 ADR，不在各页面自行判断 locale。

覆盖：

- 所有正式 Route；
- API error mapping；
- Loading/empty/offline；
- Accessibility labels；
- Auth/password；
- Review；
- Writing/signature；
- Account delete/export；
- Store screenshots/copy。

## 规则

- `interface_locale` 与 `script_preference` 正交；
- 中文姓名原样保留；
- 课程目标仍是中文；
- 幽默包按内容 locale 选择，缺失时 neutral fallback；
- 未翻译 key 在开发报错，生产安全 fallback 并遥测。

## 测试

Key completeness、页面 smoke、长文案、小屏、动态字体、错误/无障碍、locale persistence、切换后 Session Snapshot 不被改写、Store 声明一致。

## 验收
