## 数据库

新增 `pinyin_concepts` 与必要关系；扩展 concept/skill 类型。要求：

- initial/final/syllable/tone；
- canonical、display、numbered、tone number；
- initial/final 组成引用；
- audio asset；
- curriculum version；
- published 状态；
- 强约束和唯一键；
- 公共已发布内容可读，未发布不可用于 Session。

## 内容导入

把 `pinyin-content-v1` 转换为正式 Release 数据。导入必须：

- 幂等；
- 校验全部声韵引用和合法组合；
- 校验五声；
- 校验音频 hash/许可/locale；
- 生成稳定 UUID 或稳定映射；
- 不依赖测试 Seed。

## 读音策略
