## 报告规则

每项只能是：

- PASS
- FAIL
- NOT RUN
- NOT APPLICABLE

测试存在不等于运行通过。任何关键路径依赖 Demo、手工 DB Seed、占位页或未签名构建，发布结论必须是“不建议发布”。
