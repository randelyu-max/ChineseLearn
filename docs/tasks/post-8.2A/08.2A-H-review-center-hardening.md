1. Offset Cursor → Keyset Cursor；
2. 固定 `generatedAt`；
3. 到期过滤、排序、去重和 limit 下推 SQL；
4. 避免一次读取 5001 行后 Node 全量处理；
5. 明确 Active Curriculum 选择，而不是隐式 newest published；
6. 同 content 不同 skill 的 multi-skill 合并规则；
7. `pinyin_dependency` 由真实提示依赖指标决定，不再等同全部 glyph_to_sound；
8. 真实 tone/pinyin group；
9. 旧/退役课程概念的 replacement/migration 策略；
10. SQL explain 和索引证明。
