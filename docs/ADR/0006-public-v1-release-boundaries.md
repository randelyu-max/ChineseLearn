# ADR 0006 — Public V1 发布边界

- **状态：** Accepted by supplement
- **日期：** 2026-07-24

## 决策

公开 V1 不以“自动测试通过且只剩复习占位页”为完成标准。发布候选必须具备：

- 正式 learn/review/pinyin 闭环；
- 生产课程 Release；
- 真实 UI 本地化范围；
- 账号删除和数据导出；
- 用户时区；
- 诊断起点应用；
- 中文姓名书写的诚实覆盖；
- 无占位页、无生产 Demo 导航；
- 签名构建和物理设备验证。

## UI locale 决定

公开 V1 至少完整支持 `en-US` 与 `zh-CN`。`zh-TW` 只有在 100% 运行时覆盖和人工审校完成后才可暴露。学习字形偏好独立于 UI locale。

## 内容决定

生产 V1 目标仍为 100 高频汉字、200 常用词、30 日常句型和 10 个短故事，并包含基础拼音与声调。主题必须适合 13+ 与成人。

## 账号决定

应用支持创建账号，因此发布前必须提供应用内删除入口、全账号删除、关联数据处理、本地清理和外部删除资源。数据导出必须区分服务器数据和仅本地的原始书写轨迹。

## 被否决方案

- 把占位页标记为未来功能后直接发布；
- 商店声明三种界面语言但运行时只有简体；
- 用 Support 邮箱替代普通应用内删除入口；
- 依赖测试 Seed 作为生产课程；
- 宣称任意姓名都有标准笔顺。
